# Sentimentanalyse-Einfaches-API-Projekt
Dieses Projekt demonstriert eine einfache Sentimentanalyse-Anwendung mit einer API-Schnittstelle. ✅ Backend (API-Server) – Entwickelt mit FastAPI, stellt eine REST-API zur Verfügung, die Text entgegennimmt und eine einfache Sentimentbewertung zurückgibt (positiv, negativ, neutral).
